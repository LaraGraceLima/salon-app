## 🚀 The 5-Minute Setup

### Step 1: Database (1 min)
```
1. Start MySQL in XAMPP
2. Open phpMyAdmin
3. Import server/database.sql
```

### Step 2: Backend (1 min)
```bash
cd server
npm install
npm start
```

### Step 3: Frontend (1 min)
```bash
npm run dev
```

### Step 4: Login (1 min)
```
Open http://localhost:5173
Email: admin@salon.com
Password: admin123
```

### Step 5: Explore (1 min)
- Add clients
- Create stylists
- Add services
- Manage bookings
